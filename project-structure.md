# Project structure

A quick overview of the recommended project structure for this portfolio scaffold.

- pages/ or src/pages/ - Next.js pages
- components/ - Reusable React components
- styles/ - Tailwind or CSS files
- public/ - Static assets (images, icons)
- data/ or content/ - Project and blog content
- scripts/ - Utility scripts

Adjust paths to your repo's current layout as needed.
